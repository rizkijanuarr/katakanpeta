# Backend Express MVC (be-express-mvc)

Backend Express.JS arsitektur **MVC** dengan **TypeScript**.

## 🚀 Fitur Utama
- **Express.js** v5 + TypeScript
- **PostgreSQL** (`pg`) Koneksi database
- **JWT** & **Bcrypt** Autentikasi dan keamanan
- **Winston** Pencatat log (logger)

## 📦 Instalasi

Compiler **Bun** 🥟 sebagai package manager utama, untuk menginstal dependensi cukup jalankan:

```bash
bun install
```

## 🛠 Menjalankan Server

**Mode Development (Hot-Reload dengan Nodemon):**
```bash
bun run dev
```

**Mode Production / Build:**
```bash
bun run build
bun run start
```
